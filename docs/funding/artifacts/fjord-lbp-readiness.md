# Fjord LBP readiness checklist

## Current fit

Fjord is relevant as a liquidity support path for SINC, but it is not a substitute for grants and should not outrank Base or Uniswap funding work.

## Preconditions before any launch action

- Confirm token sale objective, treasury policy, and acceptable dilution/liquidity ranges.
- Confirm token, treasury, and bonding-curve addresses against canonical docs.
- Prepare sale parameters: duration, start/end pricing, token amount, quote asset, vesting, and disclosure text.
- Prepare public risk disclosure and buyer-facing documentation.
- Confirm legal/compliance/KYC requirements for the selected launch path.

## Hard-stop escalations

- Any wallet signature
- Any fund movement
- Any binding launch configuration
- Any KYC or legal acceptance

## Practical next step

Build the evidence and parameter pack first; only then consider a dry-run or platform submission review.
