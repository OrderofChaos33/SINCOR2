# Funding Operations Hub

This directory tracks non-dilutive funding, liquidity support, sponsorship, and ecosystem assistance opportunities for SINCOR / SINC / DAE.

## Operating rules

- Use verifiable project evidence only.
- Prefer official program URLs and shipped-work-based programs.
- Log every cycle, artifact, blocker, and escalation.
- Escalate wallet signatures, KYC, legal paperwork, and final high-stakes submissions.

## Structure

- [`opportunities.json`](./opportunities.json) — current scored opportunity registry.
- [`cycles/`](./cycles/) — dated execution logs and reflections.
- [`artifacts/`](./artifacts/) — reusable nomination, application, and readiness materials.
