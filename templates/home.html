→ TOA

Temporal Optimization Agent

TOA (E-toa-44) is the 44th agent — a Director archetype that sits above task routing and answers: "Given current state and objectives, which actions should we take next — and in what order?" It runs a continuous forecast → simulate → collapse pipeline with recursive self-improvement.

TOAOrchestrator ├─ KernelForecaster → Nadaraya-Watson kernel + Monte Carlo paths ├─ MonteCarloSimulator → scores revenue, risk, timeline, compliance, governance ├─ WFCCollapser → wave-function-collapse path selection → ranked actions └─ RollingFeedbackAgent → ingests outcomes (verticals, on-chain, marketplace) → next forecast

How TOA decides

Time-series context in → forecast N paths → simulate each against weighted objectives → collapse to top-k action dispatches → optional auto-route to the best-matching swarm agent.

Feedback loop

After execution, call ingest_feedback() with vertical results, settlement events, or governance outcomes. The next run incorporates that signal automatically.

Configurable via env

TOA_SIMULATION_DEPTH, TOA_COLLAPSE_THRESHOLD, TOA_OBJECTIVE_WEIGHTS, and more. Persistent session state via TOA_STATE_PATH.
