## PAYPAL SETUP CHECKLIST

1. **Navigate to:** PayPal Developer Dashboard → My Apps & Credentials → Live
2. **Find your app:** The one with Client ID starting with Ac0_uwVreyKj...
3. **Click "Edit"** to modify app settings
4. **Enable Features:**
   - ✓ Accept payments
   - ✓ Advanced Credit and Debit Card Payments
   - ✓ Subscriptions (for monthly plans)
   - ✓ Payouts (optional, for future affiliate program)

5. **Add Return URLs:**
   - http://localhost:5000/payment/success
   - http://localhost:5000/payment/cancel
   - https://sincor-production.up.railway.app/payment/success
   - https://sincor-production.up.railway.app/payment/cancel

6. **Create Products:**
   - Go to PayPal Business Dashboard
   - Products & Services → Create Product
   - Add each product using the templates above

7. **Verify & Test:**
   - Save all changes
   - Wait 2-3 minutes for propagation
   - Test with the $0.01 test price we set up
